# Intruções:

Para executar o trabalho basta utlizar o terminal e digitar "./node_modules/.bin/cypress run". Após a finalização do processo, o relatório será disponibilizado na pasta 
cypress\reports e para abri-lo basta clicar com o botão direito sobre o arquivo "index.html" e selecionar a opção "Reveal in File Explorer" ou colocar o mouse sobre ele 
e precionar Shift+Alt+R. Selecione o arquivo de mesmo nome e ecolha o seu navegador de preferência para executá-lo.
